Bad Code Smell
Code smell refers to any symptom in the source code of a program that possibly indicates a deeper problem. They are not bugs or errors but are tangible and observable violations of code design/development fundamentals that could eventually lead to poor code quality and technical debt. Code smells are subjective and can vary based on context, but they often suggest that the code could be improved in terms of readability, maintainability, or efficiency.
